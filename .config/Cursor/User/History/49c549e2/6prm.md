# Temperatura de ebullicion ( $Te$ )
$$ Te  = TeA +  \triangle Te$$
**En donde:**
   - $Tea$ = Temperatura de ebullicion del solvente puro 
   - $\triangle Te$ = Variacion del $Te$
# Variacion de la $Te$ ($Tea$)
$$\triangle Te = m.Kc $$
**En donde:**
   - $m$ = Molaridad 
   - $Kc$ = Constante ebulliscopica
